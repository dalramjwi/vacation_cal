# 터미널 휴가 관리 프로그램

Node.js로 만든 터미널(CLI) 기반 휴가 관리 프로그램입니다.

## 기능

- **휴가 설정 초기화**: 입사일을 기준으로 휴가 설정을 시작합니다.
- **휴가 사용 기록**: 시간 단위로 사용한 휴가를 기록합니다.
- **휴가 현황 조회**: 남은 휴가와 사용한 휴가를 한눈에 확인합니다.

## 설치 방법

1.  **Node.js 설치**: 이 프로그램을 사용하려면 [Node.js](https://nodejs.org/)가 설치되어 있어야 합니다.
2.  **전역(Global) 설치**: 터미널에 다음 명령어를 입력하여 프로그램을 전역으로 설치합니다.

    ```bash
    npm install -g .
    ```

    이제 터미널 어디에서든 `vacation` 명령어를 사용할 수 있습니다.

## 사용법

### 1. 초기화 (`init`)

가장 먼저 `init` 명령어로 입사일을 등록해야 합니다.

```bash
vacation init
```

터미널에 입사일을 `YYYY-MM-DD` 형식으로 입력하면 설정이 완료됩니다. 설정 파일은 `~/.vacation/vacation.json`에 저장됩니다.

### 2. 휴가 사용 기록 (`add`)

휴가를 사용했다면 `add` 명령어로 기록할 수 있습니다.

```bash
vacation add
```

사용한 날짜와 시간을 순서대로 입력하면 휴가 사용 내역이 기록됩니다.

### 3. 현황 조회 (`status`)

현재 휴가 현황이 궁금하다면 `status` 명령어를 사용하세요.

```bash
vacation status
```

다음과 같이 생성된 월차, 사용한 휴가, 남은 휴가를 상세하게 보여줍니다.

```
[휴가 현황]

입사일: 2025-05-20
생성된 월차: 11일 (88시간)
사용한 휴가: 14시간
남은 휴가: 9일 6시간 (74시간)
```

### 4. 도움말 (`help`)

명령어 사용법이 기억나지 않을 때는 `help`를 입력하세요.

```bash
vacation help
```
